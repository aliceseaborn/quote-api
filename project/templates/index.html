Welcome

This is a stock quote API. It provides free quotes for publically-traded American equities. It was created to aid in long-term financial research due to its focus on providing holistic data about the companies in question. For example, while many alternatives provide data on stock price, few low-prices commercial APIs provide datapoints for dividend dates, financial targets, etc. We provide these data for free by screen-scraping resources, compiling stock meta-data, and presenting them RESTfully to feed your research efforts.

Expensive and superior alternatives exist which provide near real-time data. If you are interested in feeding an automated trading algorithm, for instance, we strongly suggest you retain their services and leave us behind. This API is not designed for and should not be judged by its speed. On the other hand, if your research simply needs relevant data, this API might come in handy.

Browse the docs to learn about the support API versions and resources. If you have questions, you are free to open a ticket with our GitHub project and ask us.
